== Description ==

This plugin allows you to display the latest Yahoo Answers directly on your blog - Using any search query on Yahoo Answers you wish!

Simply enable the plugin through your admin panel, edit the JR Answers settings in the Settings --> JR Answers menu and enable the widget through the Appearance --> Widgets menu! The latest Yahoo Answers posts will be displayed on your blog!

== Installation ==

1. Upload `jr-answers.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Answers Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets menu.
